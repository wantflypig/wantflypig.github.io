面向对象之基础

什么是面向对象

  • 面向对象是使用对象进行开发(指挥者)
  • 面向对象是面向过程的封装
  • 面向过程是按照过程是按照过程一步一步执行(执行者)
  • 面向对象是将执行者变成指挥者

面向对象的特性

抽象性

  1. 抽取核心数据用来描述
  2. 在特定环境下才知道这个对象的真正意义

封装性

  • 将数据(属性)和功能(方法)进行封装
  • 成员是简单数据类型就是属性,成员是函数 就是方法
  • 方法就是过程的封装

继承性

  • 将别人的方法和属性拿来自己使用,并且成为自己的成员
  • 传统继承基于模板

  • js继承基于对象

    mix混入方法实现继承(JQ中是extend方法)
function mix(obj){
    var newObj = {};
    for(var k in obj){
        newObj[k] = obj[k];
    }
    return newObj;
}
